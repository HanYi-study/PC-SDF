# The application process of PC-SDF  
<img width="731" height="712" alt="image" src="https://github.com/user-attachments/assets/00515112-0319-4fe4-8489-eec1e03205e1" />

The PC-SDF framework implements the GSR and PDEC modules within a unified pipeline. For the DEM and point cloud registration process (PDEC), the complete and reproducible scripts are currently under code review and cleanup. The full source code and sample datasets will be made publicly available upon the official acceptance or publication of the related manuscript.
